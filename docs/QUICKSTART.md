# QUICKSTART

## 1. 安装

### Windows 入口

在项目根目录双击运行 `bootstrap.bat`。

### 纯 WSL 入口

```bash
./install.sh
```

## 2. 启动

```bash
./start_8080_toolhub_stack.sh start
```

## 3. 打开网页

浏览器访问 `http://127.0.0.1:8080`

## 4. 查看状态

```bash
./start_8080_toolhub_stack.sh status
```

## 5. 查看日志

```bash
./start_8080_toolhub_stack.sh logs
```

## 6. 停止服务

```bash
./start_8080_toolhub_stack.sh stop
```
